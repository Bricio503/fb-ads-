Connect your Meta account

Pull spend, leads and per-ad performance straight from the Meta Marketing API. Two short setup steps, then it's one click forever after.

1
Get an access token with ads_read permission. Fastest test route: Graph API Explorer → pick your app → add ads_read → Generate. For one that doesn't expire, create a System User token in Business Settings → Users → System Users.
2
Grab your Ad account ID from Ads Manager (top-left account picker — looks like act_1234567890).
3
Open Settings (gear, top right), paste both in, set your avg assignment fee, hit Save & refresh.
